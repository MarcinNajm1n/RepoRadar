# RepoRadar Roadmap

## Current MVP

- Local Next.js + SQLite app.
- Daily GitHub scanning.
- Deterministic trend scoring.
- Polish summaries, reports, and ideas.
- Radar dzisiaj decision workflow.
- Action queue.
- Evidence-backed business candidates.
- Windows/Discord notifications.
- Daily/weekly markdown reports.
- CSV export and maintenance actions.

## Near-Term

- Improve first-run onboarding and empty states.
- Add more UI coverage after installing Playwright browsers.
- Add screenshots to `docs/screenshots/` from real seeded or real local data.
- Add richer filters for opportunity score and action item status.
- Add print-specific CSS for report views.

## Later

- Optional Tauri wrapper.
- Optional PostgreSQL/Supabase migration path, without making it an MVP dependency.
- More source adapters where cost and terms are acceptable.
- Better report search and saved views.
- More notification controls, including quiet hours if needed.

## Out Of Scope For MVP

- Public SaaS.
- User accounts/auth.
- Billing.
- Team accounts.
- Mass repository cloning.
- Browser extension.
- Mobile app.
- Automatic pull requests to discovered repositories.
