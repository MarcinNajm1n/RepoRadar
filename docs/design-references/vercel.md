# Vercel Reference Notes

Role: console hierarchy, deployment-style clarity, and restrained operational UI.

## Useful Patterns

- Minimal page chrome around high-signal operational content.
- Crisp panel boundaries with understated elevation.
- Compact status and metadata groupings.
- Clear primary action placement without oversized hero treatment.

## RepoRadar Translation

- Keep the top bar as an operational command strip: scan, briefing, export, weekly report.
- Use graphite-neutral surfaces with small elevation jumps.
- Keep metrics tabular and aligned.
- Prefer direct status language over decorative visuals.

## Boundaries

- Do not copy Vercel's brand expression or monochrome-only feel.
- RepoRadar needs warmer semantic status colors because repository evaluation includes uncertainty, risk, and opportunity.
